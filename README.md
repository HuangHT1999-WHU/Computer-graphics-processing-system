# Computer graphics processing system
Graphics processing system developed based on computer graphics
This is the course design for the 2020 "Computer graphics" course internship.
The main content is a variety of graphics generation algorithms, change algorithms, area filling algorithms and other commonly used computer graphics codes.
The author is Huang Hongtian, a 2018 undergraduate student majoring in geographic information systems, School of Remote Sensing and Information Engineering, Wuhan University
